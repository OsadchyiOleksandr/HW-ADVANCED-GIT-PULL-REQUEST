В проекті має бути реалізовано функціонал конвертації фаренгейтів в цельсії, цельсіїв в фаренгейти, проведено роботу з pull request. Розробку ведіть під Git та GitHub. Зверніть увагу на створення комітів, формування повідомлень комітів, гілкування, створення та видалення pull request.

(1) Cтворіть проект Converter на локальній машині через IntelliJ IDEA (IDE).

(2) В гілці master , додайте функціонал виведення інформації про проект/додаток в консоль.

(3) Зробіть початковий коміт.

(4) Створіть репозиторій на GitHub.

(5) Запуште зміни в репозиторій на GitHub.

(6) Створіть гілку develop з гілки master.

(7) В гілці develop додайте функціонал конвертації фаренгейтів в цельсії.

(8) Зробіть коміт цих змін.

(9) Запуште ці зміни на GitHub.

(10) На GitHub створіть pull request для можливості злиття гілки develop в гілку master.

(11) Створіть гілку feature-cls-frt з гілки develop.

(12) В гілці feature-cls-frt додайте функціонал конвертації цельсіїв в фаренгейти.

(13) Зробіть коміт цих змін.

(14) Запуште ці зміни на GitHub.

(15) Створіть pull request для можливості злиття гілки feature-cls-frt в гілку develop.

(16) На GitHub, здійсніть злиття гілки feature-cls-frt в гілку develop.

(17) Оновіть гілку develop в локальному репозиторії.

(18) Видаліть гілку feature-cls-frt в локальному та віддаленому репозиторіях.

(19) На GitHub, закрийте pull request щодо можливості злиття гілки develop в гілку master.

(20) Посилання на свій GitHub репозиторій зазначте в LMS.
